## 📋 Dars Rejasi

Bu darsda biz **Anvarning hikoyasini** boshlaymiz. U Samarqandda kichik non do'koni ochadi va biznesini **oddiy daftar** bilan boshqaradi. Biz uning qanday muammolarga duch kelishini ko'ramiz.

---

## 🏪 Anvarning Hikoyasi Boshlanadi

### 2020-yil, Samarqand

Anvar — 28 yoshli yigit. U yillar davomida novvoylikni o'rganib, nihoyat o'z do'konini ochishga qaror qildi.

```
┌──────────────────────────────────────────┐
│       🏪 "ANVAR NON" DO'KONI             │
│                                          │
│   📍 Samarqand, Registon ko'chasi        │
│   👨‍🍳 1 ta novvoy (Anvarning o'zi)       │
│   👦 1 ta yordamchi                       │
│   📦 Mahsulotlar: Non, Patir, Somsa      │
│   💰 Boshlang'ich kapital: 15,000,000    │
│                                          │
└──────────────────────────────────────────┘
```

Anvar do'konini ochdi. Endi uni **boshqarish** kerak. Uning qo'lida bitta vosita bor — **daftar va ruchka**.

## 📓 Daftar Bilan Boshqarish

### Anvar 3 ta daftar yuritadi:

```
📓 1-DAFTAR: SAVDO DAFTARI
   Har kuni qancha non sotildi, qancha pul tushdi

📓 2-DAFTAR: XARAJAT DAFTARI
   Nima sotib olindi, qancha pul sarflandi

📓 3-DAFTAR: QARZ DAFTARI
   Kim qarz oldi, qancha, qachon qaytaradi
```

## 🚨 Muammolar Boshlandi!

### 🔴 Muammo #1: Hisob-kitob xatolari

```
  Vaziyat:
  Anvar kechqurun charchab uyga keldi.
  Savdo daftariga yozayotganda 280,000 o'rniga 28,000 yozib qo'ydi.
  
  Natija:
  Oy oxirida 252,000 so'm "yo'qoldi".
  Anvar buni 3 kun izladi — topa olmadi.
  
  ⏱️ Yo'qotilgan vaqt: 3 kun
  💰 Yo'qotilgan pul: 252,000 so'm
```

### 🔴 Muammo #2: Qarz daftari chalkashligi

```
  Vaziyat:
  Karim aka aytdi: "Men qarzimni to'ladim!"
  Anvar daftarga qaradi — Karimning sahifasi o'chirilgan 
  (yordamchi bola chizib yuborgan).
  
  Natija:
  Anvar Karim akaga ishondi. Lekin aslida 20,000 qarz bor edi.
  
  💰 Yo'qotilgan pul: 20,000 so'm
  😤 Buzilgan munosabat: Keyinchalik haqiqat aniqlanganda noqulaylik
```

### 🔴 Muammo #3: Ombor nazorati yo'q

```
  Vaziyat:
  Juma kuni 100 ta somsa uchun buyurtma keldi.
  Anvar ishga kirishdi — lekin tuxum tugagan ekan!
  
  Natija:
  Soat 5 da bozorga yugurdi. 
  Bozor narxi tegirmon narxidan 30% qimmat.
  Buyurtma 2 soat kechikdi.
  
  ⏱️ Yo'qotilgan vaqt: 2 soat
  💰 Qo'shimcha xarajat: 30% ortiq to'lov
  😤 Mijoz noroziligi: Restoran endi ishonchsiz qaradi
```

### 🔴 Muammo #4: Oylik hisobot — dahshat

```
  Vaziyat:
  Oy oxirida Anvar o'tirdi — "Bu oy qancha foyda qildim?"
  
  Jarayon:
  1. 30 kunlik savdo yozuvlarini bir-bir qo'shdi — 3 soat
  2. 30 kunlik xarajatlarni qo'shdi — 2 soat
  3. Qarz daftarini tekshirdi — 1 soat
  4. Raqamlar to'g'ri chiqmadi — qayta hisobladi — 2 soat
  5. Yana to'g'ri chiqmadi — yana hisobladi — 1 soat
  6. "Yaqin chamasiga" chiqardi — charchadi
  
  ⏱️ Jami: 9 soat (deyarli 1 ish kuni!)
  📊 Aniqlik: ~85% (15% xatolik)
```

### 🔴 Muammo #5: Daftar yo'qoldi!

```
  ⚡ ENG DAHSHATLI VOQEA ⚡
  
  Bir kuni yordamchi bola daftarni tozalik paytida ho'l qildi.
  Xarajat daftarining 2 haftalik yozuvlari o'qib bo'lmaydi.
  
  Natija:
  - 2 haftalik xarajat ma'lumotlari butunlay yo'qoldi
  - Soliq hisobotini to'g'ri tayyorlab bo'lmadi
  - Anvar taxminan hisobladi — soliq inspektori savollar berdi
  
  💰 Potensial jarima xavfi
  😰 Stress darajasi: YUQORI
```

## 💡 Anvarning "Aha!" Momenti

Bir kuni Anvar do'stining kompyuterida **Excel** dasturini ko'rdi.

```
  Do'sti:  "Anvar, men barcha hisob-kitobimni shu dasturda qilaman.
            Formulalar o'zi hisoblab beradi. Grafik chizadi.
            Qidirish — bir soniya."
  
  Anvar:   "Lekin men kompyuterni bilmayman..."
  
  Do'sti:  "O'rgataman. 1 haftada o'rganib olasan."
  
  Anvar:   *Daftariga qaradi... keyin kompyuterga qaradi...*
           "Boshladik!"
```

> [!NOTE]
> Bu — biznes evolyutsiyasining **birinchi qadami**. Anvar tushundi: biznes o'sishi bilan boshqaruv vositalari ham o'sishi kerak. Daftar — 5-10 ta mahsulot va 20-30 ta kunlik xaridor uchun yetarli. Lekin 200+ mijoz, 100+ kunlik buyurtma, 10+ ta xom ashyo turlari bilan ishlashda daftar **to'xtab qoladi**.

---

## 📈 Qachon Daftardan O'tish Kerak?

Agar quyidagilardan **3 tasi** sizga tegishli bo'lsa — daftardan o'tish vaqti kelgan:

| # | Belgi | Anvar uchun? |
|---|-------|-------------|
| 1 | Kuniga 50 dan ortiq tranzaksiya | ✅ Ha (150+) |
| 2 | Hisob-kitobda xatoliklar tez-tez | ✅ Ha |
| 3 | Oylik hisobot tayyorlash 4+ soat | ✅ Ha (9 soat!) |
| 4 | Qarzlarni kuzatish qiyinlashgan | ✅ Ha |
| 5 | Omborda nima borligini bilmaysiz | ✅ Ha |
| 6 | 2+ ishchi bor | ✅ Ha |
| 7 | Soliq hisoboti qiyin | ✅ Ha |

**Anvar uchun natija: 7/7 — Daftardan o'tish SHOSHILINCH!** 🚨

4. **Agar siz Anvarning o'rnida bo'lsangiz, birinchi nima qilgan bo'lardingiz?**

## 💡 Asosiy Xulosalar

| Xulosa | Tushuntirish |
|--------|-------------|
| Daftar — boshlang'ich uchun yetarli | 1-2 kishi, 20-30 mijoz, oddiy operatsiyalar |
| Biznes o'sganda daftar to'xtab qoladi | Xatoliklar, sekinlik, ma'lumot yo'qolishi |
| Evolyutsiya tabiiy jarayon | Har bir biznes shu bosqichdan o'tadi |
| Vaqtida o'tish muhim | Kech qolish = ko'proq yo'qotish |

> **Keyingi darsda** Anvar Excel bilan tanishadi. Formulalar, jadvallar, grafiklar — yangi dunyo ochiladi. Lekin... yangi dunyo — yangi muammolar ham olib keladi.
